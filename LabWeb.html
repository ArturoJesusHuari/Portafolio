LabWeb

Los laboratorios prácticos del curso ISO93A-T 'Desarrollo de Aplicaciones Web' de la Facultad de Ingeniería de Sistemas. Estos laboratorios están diseñados como guías prácticas para complementar la enseñanza teórica del curso. En este entorno, los estudiantes tienen la oportunidad de aplicar directamente los conceptos y técnicas aprendidas en clase a través de ejercicios prácticos.

Ejercicio 3

Enunciado

Diseñar un loader (animación de precarga) como un anillo que gira con animación infinita.

Código

HTML

<!DOCTYPE html>
                <html lang="es">
                <head>
                  <meta charset="UTF-8">
                  <meta name="viewport" content="width=device-width, initial-scale=1.0">
                  <title>Pantalla de Carga</title>
                  <link rel="stylesheet" href="loader.css">
                </head>
                <body>
                  <div class="loader"></div>
                </body>
                </html>

CSS


                body {
                  margin: 0;
                  padding: 0;
                  display: flex;
                  justify-content: center;
                  align-items: center;
                  height: 100vh;
                  background-color: #f0f0f0;
                }
                
                .loader {
                  border: 4px solid rgba(0, 0, 0, 0.1);
                  border-top: 4px solid #13c6d3f8;
                  border-radius: 50%;
                  width: 150px;
                  height: 150px;
                  animation: spin 0.5s linear infinite; /* Animacion infinita */
                }
                
                @keyframes spin {
                  0% {
                    transform: rotate(0deg);
                  }
                  100% {
                    transform: rotate(360deg);
                  }
                }
